# Theme Hub Roadmap

## Now

- Discover installed dashboard themes.
- Discover plugin-provided `theme/*.yaml` files.
- Install a selected plugin theme into the active Hermes home.
- Activate installed or built-in themes through dashboard config.
- Avoid exposing personal local paths in UI and docs.

## Next

- Add richer theme previews.
- Show validation warnings for incomplete theme YAML.
- Add import/export helpers for theme packs.
- Surface theme screenshots when plugin packages provide them.

## Later

- Support official theme signatures if Hermes publishes a trust root.
- Support community review metadata for shared theme packs.
- Integrate Theme Hub readiness into Hackathon Hub submission checks.
